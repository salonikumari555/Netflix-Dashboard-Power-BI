# 🎬 Netflix Content Analysis Dashboard

An interactive Netflix Content Analysis Dashboard built using Microsoft Power BI to analyze Netflix's movies and TV shows. The dashboard transforms raw data into meaningful visualizations, helping users explore content trends, genres, ratings, release years, countries, and more.

## 📌 Project Overview

The objective of this project is to perform data analysis on Netflix's content library and present key insights through an interactive Power BI dashboard. It enables users to filter and explore the dataset for better understanding of Netflix's content distribution and trends.

## 🎯 Objectives

* Analyze Netflix's content library.
* Compare Movies and TV Shows.
* Identify the most popular genres and ratings.
* Explore content distribution across countries.
* Analyze release year trends.
* Build an interactive dashboard for data-driven insights.

## 🛠️ Tools & Technologies

* Microsoft Power BI
* Power Query
* DAX (Data Analysis Expressions)
* Microsoft Excel / CSV Dataset

## 📊 Dashboard Features

* Total Movies and TV Shows
* Content Distribution by Type
* Content by Rating
* Genre Analysis
* Country-wise Content Distribution
* Release Year Trends
* Interactive Filters (Slicers)
* KPI Cards and Visualizations

## 📈 Key Insights

* Movies make up the majority of Netflix's content.
* Drama, Comedy, and Documentary are among the most common genres.
* Netflix content has grown significantly over recent years.
* The United States contributes the largest share of available titles.
* TV Shows represent a smaller portion of the overall catalog compared to Movies.
